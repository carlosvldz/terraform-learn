# Terraform

Practice repository with Terraform.

## Run any project with:
```bash
# Download provider plugins
terraform init
# Validate syntax
terraform validate
# Plan
terraform plan
# Run
terraform apply
# Run (without approve)
terraform apply -auto-approve
# Destroy provisioned infrastructure
terraform destroy
```
